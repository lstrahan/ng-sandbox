The 'ags-hmi-library' package is not included in this git repo, but is required to build this project.
You must build the 'ags-hmi-library' from the AGS-HMI project using the 'run npm build-lib' script.
Then copy the .tgz file from the /dist/ags-hmi-library folder to the root folder of this project.


To run the web socket server...  node projects\sandbox\webSocketServer.js
